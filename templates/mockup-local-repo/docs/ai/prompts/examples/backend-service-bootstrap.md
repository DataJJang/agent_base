# Backend Service Bootstrap Example

```text
Run a project bootstrap interview for a new backend service repository.

Repository: my-backend-service
Branch: dev
Goal: bootstrap a new backend service with API, DB, and deployment readiness
Target environments: local, dev, stg
Preferred project family: backend-service
Expected runtime roles: api
Out of scope:
- business endpoint implementation

Ask the user for:
- project name and purpose
- API contract and external integration profile
- DB, cache, auth, and deployment choices
- logging and runtime profile policy
- target OS and service startup mode

Then produce a normalized project generation spec.
```
