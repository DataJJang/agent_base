# Agent Failure Review Checklist

## 1. 기본 정보

- [ ] 저장소명과 브랜치를 적었다
- [ ] 작업 요청을 한 줄로 정리했다
- [ ] 기대 결과와 실제 결과를 분리해서 적었다

## 2. 실패 유형 분류

- [ ] 규칙 부족인지 확인했다
- [ ] 문서 드리프트인지 확인했다
- [ ] prompt 부족인지 확인했다
- [ ] template 또는 scaffold 부족인지 확인했다
- [ ] validation 누락인지 확인했다
- [ ] 도구 한계인지 확인했다

## 3. 영향도

- [ ] 코드 영향
- [ ] 설정 영향
- [ ] DB 영향
- [ ] 문서 영향
- [ ] 운영 영향
- [ ] 위험 명령 또는 위험 SQL 여부

## 4. 환류 위치

- [ ] `AGENTS.md`
- [ ] `docs/ai/*`
- [ ] `prompts/*`
- [ ] `templates/*`
- [ ] `scripts/*`
- [ ] `checklists/*`

## 5. 완료 조건

- [ ] root cause를 적었다
- [ ] 재발 방지 조치를 적었다
- [ ] 필요한 문서 또는 템플릿을 수정했다
- [ ] 같은 실패가 줄었는지 확인 가능한 검증 기준을 적었다
