# Mobile App Bootstrap Example

```text
Run a project bootstrap interview for a new mobile application repository.

Repository: my-mobile-app
Branch: dev
Goal: bootstrap a mobile app project for Android and iOS
Target environments: local, dev, stg
Preferred project family: mobile-app
Expected runtime roles: client
Out of scope:
- app store release execution

Ask the user for:
- project name and purpose
- Flutter, React Native, or native stack choice
- target platforms and minimum OS support
- backend dependency and authentication type
- local build, test, smoke, and deployment direction

Then produce a normalized project generation spec.
```
