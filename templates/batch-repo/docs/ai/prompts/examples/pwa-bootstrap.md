# PWA Bootstrap Example

```text
Run a project bootstrap interview for a new PWA repository.

Repository: my-pwa-app
Branch: dev
Goal: bootstrap a PWA project with offline and installable behavior
Target environments: local, dev, stg
Preferred project family: pwa
Expected runtime roles: frontend
Out of scope:
- backend-service implementation

Ask the user for:
- project name and purpose
- offline/cache/installability requirements
- external API dependency
- datastore and cache necessity
- deployment type and startup mode
- logging approach and target browsers or OS

Then produce a normalized project generation spec.
```
