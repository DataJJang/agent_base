# Mockup Local Bootstrap Example

```text
Run a project bootstrap interview for a new local-only mockup repository.

Repository: my-local-mockup
Branch: dev
Goal: bootstrap a local-only mockup or demo repository
Target environments: local
Preferred project family: mockup-local
Expected runtime roles: frontend or client
Out of scope:
- production deployment
- external production integrations

Ask the user for:
- project name and purpose
- local-only versus shareable demo goal
- lightest acceptable stack
- whether DB, cache, auth, or deployment are needed at all
- how smoke validation should be documented

Then produce a normalized project generation spec.
```
