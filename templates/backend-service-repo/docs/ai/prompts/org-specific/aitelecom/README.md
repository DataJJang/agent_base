# AITelecom Prompt Pack

이 디렉토리는 AITelecom 계열 저장소에서만 쓰는 선택형 prompt pack 예시다.

사용 기준:

- 공통 prompt만으로 부족하고, 실제 저장소 경로와 운영 문서 구조가 고정돼 있을 때
- `aitelecom-admin`, `aitelecom-backend`, `aitelecom-service-api`, `aitelecom-batch`, `aitelecom-receiver*`처럼 이미 구조가 안정된 저장소를 다룰 때

포함 문서:

- `aitelecom-admin.md`
- `aitelecom-backend.md`
- `aitelecom-service-api.md`
- `aitelecom-batch.md`
- `aitelecom-receiver.md`

기본 원칙:

- 먼저 공통 규칙인 `AGENTS.md`, `docs/ai/README.md`, `docs/ai/prompts/README.md`를 따른다.
- 이 디렉토리의 문서는 공통 규칙을 대체하지 않고, 조직 특화 overlay만 제공한다.
